# AI Life Coach 🤖

## Problem
Students face stress, confusion, and lack of guidance.

## Solution
AI Life Coach provides instant suggestions for real-life problems.

## Tech Stack
- HTML, CSS, JavaScript
- FastAPI

## Features
- Instant response
- Clean UI
- Real-life problem solving

## Future Scope
- Voice input
- Emotion detection
- Chat history

## How to Run
1. Run backend:
   uvicorn main:app --reload

2. Open frontend:
   index.html (Live Server)
